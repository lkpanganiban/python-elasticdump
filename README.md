# Python ELASTICDUMP
A Python implementation of elasticdump. 

## Requirements
- Python 3.6+
- Requests
- Fire
- TQDM

## Setup
1. Install Python dependencies using `requirements.txt`
2. Run `python setup.py develop`.

## Usage
```
`elasticdump esdump --input=<url-with-index> --output=<output-file> --size=<size>`
```
